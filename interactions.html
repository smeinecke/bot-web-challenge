About These Tests

The interaction-based tests analyze user behavior during form completion:

  • Mouse Tracking: Path analysis, straight-line detection, movement patterns
  • CDP Mouse Leak: Detects Chrome DevTools Protocol by analyzing screenX/Y vs clientX/Y discrepancies
  • Typing Speed: Characters per second analysis (>15 CPS is suspicious)
  • Form Timing: Completion time analysis with focus event tracking
  • Paste Detection: Identifies paste-only input without keystrokes

Tracking begins when the page loads and continues until form submission.
