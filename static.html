About These Tests

The static fingerprinting tests analyze browser properties that are immediately available without requiring user interaction:

  • navigator.webdriver: Direct automation indicator
  • WebGL Analysis: Renderer and vendor checks for software rendering
  • Chrome Object: Validates window.chrome consistency
  • CDP Markers: Detects Chrome DevTools Protocol automation
  • Worker Context: Compares values between main and worker contexts
  • Screen Resolution: Checks for headless Chrome defaults

These tests run automatically when the page loads.
